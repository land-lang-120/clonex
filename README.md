# CloneX Studio

> Portfolio central de lancement pour toutes les apps. Landing pages + installeurs PWA.

![Statut](https://img.shields.io/badge/statut-prod-green)
![Type](https://img.shields.io/badge/type-portfolio-blue)

## 📦 Contenu

- `index.html` — page d'accueil CloneX Studio
- `clonex.html` — page de redirection avec PWA auto-install prompt
- Landing pages pour chaque app déployée
- `server.js` — serveur Node.js léger

## 🚀 Quickstart

```bash
node server.js  # → http://localhost:3000
```
